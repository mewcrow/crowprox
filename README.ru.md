# CrowProx

Это расширение позволяет проксировать трафик браузера или его отдельных вкладок через прокси-серверы.
На данный момент тестировалось в chromium через локальный прокси (hiddify на порту 12334).

## Возможности

- **Проксирование отдельных вкладок**
  - При клике на иконку расширения либо добавление в список в настройках.
- **Переключение режимов прокси**
  - _Fixed Servers_: Проксирование всего трафика, за исключением указанных в списке хостов.
  - _OnRequest (в разработке)_: Выборочное проксирование только определенного списка хостов.
- **Гибкая конфигурация**
  - Поддержка протоколов `http`, `https`, `socks4`, `socks5` и `quic`.
  - Возможность редактирования списка исключений.

## Установка и разработка

Проект использует фреймворк [WXT](https://wxt.dev/) для разработки браузерных расширений.

```bash
pnpm install
pnpm dev
```
